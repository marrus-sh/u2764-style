Go­Home Style

Last updated : 07 April 2019

This style was used for my old website ; the initial design was committed on 06 March 2017, with final edits beïng made on 14 September of that year.

The expected page style is one like the following : 

<header>
	<h1>Site Title</h1>
	<aside>Spash quote ( optional )</aside>
</header>
<main>
	<!-- Site content -->
</main>

The preferred fonts are Catamaran and Cormorant Garamond ; otherwise a sans­‑serif ( italics not required ) with weights of { 100, 800 } and a serif ( with italics ) with weights of { 300, 500, 700 } are ideal. It looks fine with default fonts though.
